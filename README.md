# ⚡ Voltify Electronics

Voltify Electronics is a dynamic e-commerce web application designed to provide a seamless online shopping experience for electronic products. It includes user authentication, product management, and a responsive user interface.

---

## 🚀 Features

- 🔐 User Registration & Login System  
- 🛍️ Product Listing with Categories  
- 🛒 Add to Cart & Checkout Functionality  
- 🧑‍💼 Admin Panel (CRUD Operations)  
- 💬 Dynamic Testimonial Carousel  
- 📱 Fully Responsive Design  

---

## 🛠️ Technologies Used

- PHP  
- MySQL  
- HTML5  
- CSS3  
- Bootstrap  
- JavaScript
